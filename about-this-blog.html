About This Blog

Sat 19 August 2017
By Andrew Kail

Just a short introduction, this blog is my attempt at expanding my knowledge and experiment with full stack engineering. I will cover a lot of different topcis on hardening, security, containers, databases, monitoring, orchestration, web servers, cloud services, home automation, and continuous delivery while trying to apply some of it to the development of this blog and other applications I am working on.

As of this post this blog is built on the following stack: * Python 3.6 and Flask web framework * sqlite database (possibly postgres) * Centos 7 * Nginx

The source code is available here on github. The stack could sightly change in the future and I will make release posts as that happens.

Thank you for visiting! Hopefully comments will be implemented soon so until then you can email me to leave some feedback.
